## 学习

### Java

1. 巩固 Java 基础语法使用（推荐阅读 [Java 编程入门](https://www.ibm.com/developerworks/cn/java/intro-to-java-course/index.html) ）
2. 学习 Java 方法
3. 了解几个常用类：
	* Number & Math 类
	* Character 类
	* String 类
	* StringBuffer 和 StringBuilder 类
	* 日期时间
	* 正则表达式
	* 流(Stream)、文件(File)和IO
	* Scanner 类
3. 学习 Java 异常处理

### XML

* XML 简介
* XML 用途
* XML 树结构
* XML 语法
* XML 元素
* XML 属性
* XML 验证
* XML 验证器
* XML 查看
* XML CSS
* XML XSLT
